# Security Policy

## Reporting

Please do not publish private pairing summaries, pairing key files, Android app
data, MQTT passwords, or full debug captures in public issues.

For sensitive reports, contact the maintainer privately first and include only:

- Add-on version
- Home Assistant version
- Hardware platform
- Redacted logs
- High-level reproduction steps

## Scope

This project is a read-only local telemetry bridge. Security-sensitive material
includes companion IDs, pairing summaries, pairing key files, MQTT credentials,
private captures, and any copied mobile app storage.
